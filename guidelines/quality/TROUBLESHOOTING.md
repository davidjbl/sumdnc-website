> Common issues and solutions.

## Common Issues

### Build Errors
- Clear `.next` cache
- Delete `node_modules` and reinstall
- Check TypeScript errors

### Runtime Errors
- Check browser console
- Verify environment variables
- Review server logs

## Debugging Tools

- Chrome DevTools
- React DevTools
- VS Code debugger
