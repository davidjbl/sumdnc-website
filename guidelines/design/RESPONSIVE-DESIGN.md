> Responsive design patterns and best practices.

## Mobile-First Approach

- Start with mobile layouts
- Use Tailwind breakpoints (sm, md, lg, xl, 2xl)
- Test on real devices

## Flexible Layouts

- CSS Grid and Flexbox
- Container queries
- Fluid typography
